Kullanımı Hakkında:

1-) package.json içerisindeki gerekli modüllerin kurulması.
2-) config.js içerisindeki databaseName connectURI gibi değerlerin düzeltilmesi.
3-) Botu başlat: node bot.js veya npm start

Not: 
kapUrl değeri domain adresinin değişme ihtimali düşünülerek config.js içerisinde konulmuştur. Domain değişmedikçe o değere dokunulmamalıdır.
